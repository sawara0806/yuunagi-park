# ゆうなぎ公園

SNSなど情報過多から離れて、実際に公園で過ごしたようにリラックスするためのWebサイトです。

## 遊び方

URLを開いて「入園する」を押してください。

- ↑↓ … あるく
- ←→ … よこあるき
- クリック … マウス見まわしの ON・OFF
- Q / E … 旋回
- M … 音の ON・OFF

スマホでは左下のボタンで歩き、画面をドラッグすると見まわせます。南の門から出ると退園します。
ベンチの近くでスペースキー（スマホは「すわる」ボタン）で腰かける／立つことができます。

環境音（風や鳥の声など）を流すかどうかは入園画面のチェックボックスで選べます。
入園後はPCならMキーでON・OFFを切り替えられます。スマホなど途中で選び直したい場合は、
南の門から一度退園してから入園し直してください。

## 特徴

- 一人称3D視点で歩けるドット絵の公園
- 自前のソフトウェアレンダラによる描画（three.js等のライブラリは未使用）
- ビルドなし・依存ゼロの素のJS
- 朝・昼・夕・夜の4時間帯と、春夏秋冬の4季節
- Web Audioで合成した環境音（鳥・虫・風・車の音など）
- 雨ボタンで雨の公園に切り替えられます（雨音・濡れた広場・水たまり）
- ネコが2匹います

画風はピクセルアーティスト「せたも」氏の作品を参考にしています。

## ローカルでの実行方法

このプロジェクトはビルド不要ですが、`file://` で直接開くと動かない場合があるため、
任意のHTTPサーバーで配信してください。

```
python3 -m http.server
```

起動後、表示されたURL（例: `http://localhost:8000/`）をブラウザで開いてください。

## v1/ について
